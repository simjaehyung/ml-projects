## 🔷 Polygon 모드 사용법 (3초!)

### 단계 1: Polygon 선택
```
🔷 Polygon 버튼 클릭
→ 모드 힌트 변경: "클릭: 점 추가 | ● 첫 점 근처 클릭/Enter: 완성"
```

### 단계 2: 이미지에서 점 추가
```
Canvas에서 순서대로 클릭:
  Click 1 (좌상단)
  Click 2 (중상단)
  Click 3 (우상단)
  ...
  (최소 3번 이상)
```

### 단계 3: 완성
```
방법 A: 첫 점 근처 다시 클릭 (초록 마커 표시)
방법 B: Enter 키 누르기
→ 자동 저장!
```

### 취소
```
Esc 키 누르기
→ 현재 다각형 초기화, 새로 시작
```

## ❓ FAQ (자주 묻는 질문)

### Q: Polygon이 자동 완성 안 됨
**A:** 최소 3점 이상 필요합니다.
- 우측 상단에 "3pts" 같이 점 수 표시 확인

### Q: 실수한 점을 지우고 싶어요
**A:** Esc 키 눌러서 다시 시작하세요

### Q: Box와 Polygon을 섞어 쓸 수 있나요?
**A:** 네! 자유롭게 전환 가능합니다
```
Box 모드로 자동차 박스 그리기
→ Polygon 모드로 전환
→ 균열 다각형 그리기
→ 둘 다 저장됨
```

### Q: 증강 없이 라벨만 저장하고 싶어요
**A:** 모든 증강 옵션 체크 해제하면 됩니다

### Q: 생성 개수는 몇 개가 적당한가?
**A:** 
```
테스트: 1~2개 (빠름)
일반:   5~10개 (권장)
최대:   15~20개 (시간 오래 걸림)
```
