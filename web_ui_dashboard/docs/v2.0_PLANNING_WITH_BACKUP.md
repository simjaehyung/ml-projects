### Phase 2: 백엔드 Polygon 지원 (augmentation_recipe.py)

#### 2.1 다각형 Homography 변환
```python
def transform_polygon_homography(
    points: np.ndarray,  # (N, 2)
    H: np.ndarray,        # (3, 3)
    img_w: int,
    img_h: int
) -> np.ndarray:
  """
  N개 점의 다각형을 Homography 변환.
  
  점들을 동차 좌표계로 변환 → H 행렬 곱 → 원래 좌표로 복원
  """
  N = points.shape[0]
  
  # (N, 2) → (N, 3): 동차 좌표 추가
  ones = np.ones((N, 1))
  homogeneous = np.hstack([points, ones])  # (N, 3)
  
  # (3, 3) @ (N, 3)^T = (3, N)
  transformed = (H @ homogeneous.T).T  # (N, 3)
  
  # Perspective divide: (x', y', w) → (x'/w, y'/w)
  w = transformed[:, 2]
  w = np.where(np.abs(w) < 1e-10, 1e-10, w)  # 0 나누기 방지
  
  xs = transformed[:, 0] / w
  ys = transformed[:, 1] / w
  
  # 이미지 경계 클리핑
  xs = np.clip(xs, 0, img_w)
  ys = np.clip(ys, 0, img_h)
  
  return np.column_stack([xs, ys]).astype(np.int32)
```

### Phase 3: Copy-Paste Augmentation (신규 기법)

#### 3.1 함수 구현
```python
def apply_copy_paste(
    image: np.ndarray,
    annotations: List[dict],
    background_images: List[np.ndarray],
    num_pastes: int = 3
) -> Tuple[np.ndarray, List[dict]]:
  """
  Copy-Paste Augmentation: Polygon 객체를 다른 배경에 합성.
  
  Args:
      image: 원본 이미지 (H, W, 3)
      annotations: 어노테이션 목록 (type='polygon'만 사용)
      background_images: 배경 이미지 풀
      num_pastes: 합성할 횟수
  
  Returns:
      (합성 이미지, 합성된 어노테이션)
  """
  img_h, img_w = image.shape[:2]
  
  # Polygon 객체만 필터링
  polygon_anns = [a for a in annotations if a.get('type') == 'polygon']
  
  if not polygon_anns or not background_images:
      return image, annotations
  
  # 배경 이미지 무작위 선택
  bg_idx = np.random.randint(0, len(background_images))
  bg_image = background_images[bg_idx]
  bg_h, bg_w = bg_image.shape[:2]
  
  result_image = bg_image.copy() if bg_h == img_h and bg_w == img_w else bg_image
  result_image = cv2.resize(result_image, (img_w, img_h)) if (bg_h, bg_w) != (img_h, img_w) else result_image
  result_anns = []
  
  # Polygon별로 처리
  for paste_idx in range(min(num_pastes, len(polygon_anns))):
      poly_ann = polygon_anns[paste_idx]
      points = np.array(poly_ann['points'], dtype=np.int32)
      
      # 1. Binary Mask 생성 (다각형 영역)
      mask = np.zeros((img_h, img_w), dtype=np.uint8)
      cv2.fillPoly(mask, [points], 255)
      
      # 2. 객체 추출 (마스크 적용)
      foreground = cv2.bitwise_and(image, image, mask=mask)
      
      # 3. 배경 내 무작위 위치 선택
      dx = np.random.randint(0, max(1, img_w - np.max(points[:, 0]) + np.min(points[:, 0])))
      dy = np.random.randint(0, max(1, img_h - np.max(points[:, 1]) + np.min(points[:, 1])))
      
      # 4. 객체 합성 (투명도 고려)
      pasted_mask = np.zeros((img_h, img_w), dtype=np.uint8)
      shifted_points = points + np.array([dx, dy])
      cv2.fillPoly(pasted_mask, [shifted_points], 255)
      
      # Blend 합성
      result_image = np.where(
          pasted_mask[:, :, None] > 0,
          foreground,
          result_image
      ).astype(np.uint8)
      
      # 5. 라벨 좌표 평행 이동
      new_points = points + np.array([dx, dy])
      new_points = np.clip(new_points, 0, [img_w-1, img_h-1])
      
      new_ann = dict(poly_ann)
      new_ann['points'] = new_points.tolist()
      result_anns.append(new_ann)
  
  # 기존 AABB 객체 유지
  bbox_anns = [a for a in annotations if a.get('type') != 'polygon']
  result_anns.extend(bbox_anns)
  
  return result_image, result_anns
```

### Phase 4: 백엔드 검증 업데이트 (validator.py)

```python
def validate_polygon_points(points, img_w, img_h):
  """
  Polygon points 검증.
  - 최소 3개 점
  - 각 점이 이미지 범위 내
  - 점들이 중복되지 않음
  """
  if not isinstance(points, (list, np.ndarray)):
      return False, "points는 리스트 또는 배열이어야 합니다"
  
  if len(points) < 3:
      return False, f"다각형은 최소 3개 점이 필요 (입력: {len(points)})"
  
  points_arr = np.array(points)
  
  if points_arr.shape[1] != 2:
      return False, "각 점은 (x, y) 좌표여야 합니다"
  
  # 범위 체크
  if np.any(points_arr < 0) or np.any(points_arr[:, 0] >= img_w) or np.any(points_arr[:, 1] >= img_h):
      return False, "모든 점이 이미지 범위 내에 있어야 합니다"
  
  # 면적 체크 (너무 작은 다각형 제외)
  from shapely.geometry import Polygon
  try:
      poly = Polygon(points)
      if poly.area < 100:  # 100 픽셀² 이상
          return False, f"다각형이 너무 작습니다 (면적: {poly.area})"
  except:
      return False, "유효하지 않은 다각형입니다"
  
  return True, "OK"
```

## ⚡ 성능 최적화 포인트

| 작업 | 최적화 방법 | 예상 성능 |
|------|-----------|---------|
| Homography | NumPy 벡터화 (N, 2) → (N, 3) → 행렬곱 | O(N) |
| Mask 생성 | cv2.fillPoly (C++ 구현) | O(면적) |
| Copy-Paste | np.where 벡터화 blend | O(HW) |
| 검증 | numpy 배치 연산 | O(N) |

## 📞 주의사항

⚠️ **Shapely 라이브러리**
- 다각형 면적 계산에 사용 (선택사항)
- 없어도 기본 기능 작동
- 설치: `pip install shapely`

⚠️ **메모리 사용**
- 배경 이미지 풀: 5~10개 권장
- 각 이미지 크기: ~1~5MB
- 합리적 범위 내

⚠️ **성능**
- Polygon 점 수: 100개 미만 권장
- Copy-Paste: 한 이미지당 5초 이내
- 최적화 후: 2~3초 목표
