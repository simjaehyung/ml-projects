### 3️⃣ 기본 사용법

#### 🔷 Polygon 모드 (신규)

**목표:** 자유형 다각형으로 객체 둘레를 정확히 표시

1. **"Polygon" 버튼 클릭**
   ```
   모드 전환 메시지:
   "클릭: 점 추가 | ● 첫 점 근처 클릭/Enter: 완성 | Esc: 취소"
   ```

2. **이미지 위에서 순서대로 클릭** (3번 이상)
   ```
   예: 균열(crack) 따라가기
   Click 1 (좌상단)  ──┐
   Click 2 (중상단)  ──┼─→ 다각형 형태
   Click 3 (우상단)  ──┘
   Click 4 (우하단)  ──┐
   ...
   ```

3. **완성 방법 (둘 중 하나)**
   - **첫 점 근처 다시 클릭** (초록 마커 표시)
   - **Enter 키** 누르기

4. **취소하기**
   - **Esc 키** 누르기

5. **라벨 입력 후 저장**
   - 상단 라벨 입력: "crack"
   - 💾 "저장 및 증강 실행"

## 🎬 Copy-Paste Augmentation 사용법

### 전제 조건
- ✅ 2개 이상의 이미지 업로드
- ✅ Polygon 라벨 1개 이상

### 단계별 진행

| 단계 | 동작 | 예시 |
|------|------|------|
| 1 | Polygon 모드 선택 | 🔷 Polygon 버튼 |
| 2 | 이미지 2개 이상 업로드 | car.jpg, road.jpg |
| 3 | Polygon으로 객체 표시 | 자동차 윤곽선 그리기 |
| 4 | ✅ Copy-Paste 체크 | 데이터 증강 설정 |
| 5 | 💾 저장 | "저장 및 증강 실행" |

**결과:**
```
원본 이미지      배경 이미지      합성 결과
[자동차]    +   [도로]    →   [도로 위 자동차 (다른 위치)]
│            │             │
└─ Polygon   └─ Random     └─ 라벨 자동 이동
```

## 📋 자주 묻는 질문 (FAQ)

### Q1. Polygon이 자동으로 완성되지 않습니다
**A:** 3점 이상 그렸을 때만 완성됩니다.
- 확인: 상단 우측 "점 개수" 표시
- 해결: 최소 3번 클릭 → 첫 점 근처 클릭

### Q2. 점이 잘못 들어갔어요
**A:** Esc 키를 눌러 다시 시작하세요.
```
Esc → 현재 다각형 초기화
     → 새로 그리기 시작
```

### Q3. Box와 Polygon을 섞어 쓸 수 있나요?
**A:** 네, 가능합니다!
```json
{
  "annotations": [
    {"type": "bbox", "label": "car", "x": 100, ...},
    {"type": "polygon", "label": "defect", "points": [...]}
  ]
}
```

### Q4. 증강 없이 저장만 하고 싶어요
**A:** "⚙️ 데이터 증강 설정"의 모든 체크를 해제하세요.
```
(빈 체크박스) Homography
(빈 체크박스) Poisson Noise
...
```

### Q5. 생성 개수는 어떻게 정하나요?
**A:** 슬라이더로 조정 (기본 5개)
```
슬라이더:  1 ├─────●─────| 20
         성능과 다양성의 균형점: 5~10개 추천
```

## 🐛 문제 해결 (Troubleshooting)

| 증상 | 원인 | 해결 |
|------|------|------|
| "http://localhost:5000 연결 실패" | 서버 미실행 | `python app\app.py` 다시 실행 |

## 🎯 추천 워크플로우

### 🔷 Instance Segmentation 데이터셋 구축

```
Step 1: 배경 이미지 선택
        └─ 2~3개 배경이 다른 이미지 업로드
           예: car_background_1.jpg, car_background_2.jpg

Step 2: 객체 이미지 준비
        └─ 원본 객체가 명확한 이미지
           예: car_close_up.jpg

Step 3: Polygon 라벨링
        └─ 🔷 Polygon 모드 선택
        └─ 객체 윤곽선 정확히 그리기 (5~10점)
        └─ "car_object" 라벨 입력

Step 4: Copy-Paste 증강 활성화
        └─ ✅ Copy-Paste 체크
        └─ 생성 개수: 10개
        └─ 💾 저장 및 증강 실행

Step 5: 결과 확인
        └─ project_data/augmented/ 확인
        └─ 합성 이미지 10개 자동 생성
        └─ project_data/labels/ 에 JSON 저장
```

**예상 결과:**
```
원본 1개 → 증강 10개 = 총 11개 라벨
(원본 1 + 증강 10)
```
