1조) 이미지표현과채널
발표목표
이미지를RGB/Gray/HSV 등다양한채널(색공간)로표현할수있음을이해하고, 채널변환이왜
필요한지를코드실험과함께설명한다.
발표구성
1.
RGB → Gray 와RGB 3채널을평균내어시각화하고, 두결과가왜달라지는지이유와목적을설명
한다.
2.
RGB → HSV 변환을시각화하고, “어떤작업에서어떤색공간이유리한지”를시각화및관찰하여
설명한다.
3.
RGB/Gray/HSV 외에최소2개를시각화및관찰하고, 목적/장점을말한다.
Guide Code :  이미지RGB채널시각화
3. 조별주제소개

2조) 이미지기본전처리
발표목표
이미지분석/학습전에수행하는기본전처리과정(크기·형식·스케일·채널등)을코딩실험으로확인하
고, 각전처리가왜필요한지(목적) 와결과에어떤영향을주는지를설명한다.
발표구성.
1.
여러라이브러리별차이(OpenCV vs matplotlib)를시각화를통해설명한다.
2.
데이터타입과값의범위가결과에미치는영향을이미지저장/로드실험으로확인하고, 전처리시
유의할점을설명한다.
3.
이미지resize의이유와여러가지interpolation 방법(2가지이상)에대해시각화하고설명한다.
4.
여러가지Padding 방법(2가지이상)에대해시각화하고설명한다.
Guide Code :  이미지정규화후시각화

3조) 이미지특징추출/ 필터링
발표목표
이미지에서의미있는특징을추출하는과정을이해하고, 필터링이특징추출결과에어떤영향을주는
지코딩실험으로설명한다.
발표구성.
1.
Kernel과Convolution을각각설명한다.
2.
Edge detecion의원리를수학적으로설명하고시각화한다.
3.
Corner detecion의원리를수학적으로설명하고시각화한다.
4.
Gaussian Blur와Mean Blur, Median Blur를시각화하고차이의이유를설명한다.
Guide Code : Sobel Filter를적용하여시각화

4조) 이미지강화
발표목표
이미지의시각적품질을개선하는방법을이해하고, 대표적인강화기법을코딩실험으로비교하여어
떤상황에서효과적인지를설명한다.
발표구성.
1.
명암과대조의개념을설명한다.
2.
Histogram Equalization을설명하고적용시각화하며, 적용전,후를히스토그램을통해비교한다.
3.
감마보정과이미지샤프닝을각각설명하고시각화한다.
4.
추가적인이미지강화방법을하나더설명하고시각화한다.
Guide Code : 제곱연산을통해대조강화시각화
3. 조별주제소개
8
5조) 이미지증강
발표목표
학습/분석성능을높이기위해입력이미지를다양한형태로변형하는이미지증강기법을이해하고,
증강을코딩실험으로적용하여데이터다양성이왜중요한지를설명한다. 
발표구성.
1.
증강이필요한이유를예시로설명한다.
2.
회전및원근변화의원리에대해각각설명하고시각화한다.
3.
여러가지노이즈종류에대해3가지이상설명하고차이가잘들어나도록시각화하여비교한다.
(단순Gaussian Noise나Uniform Noise는똑같은유형의Noise로생각할것)
4.
Crop과Motion Blur에대해각각설명하고시각화한다.
Guide Code : 모아레노이즈가포함된이미지생성
3. 조별주제소개

6조) 이미지합성
발표목표
두개이상의이미지를하나의장면처럼자연스럽게결합(Compositing) 하는과정을이해하고, 대표적
인Blending 기법을코딩실험으로비교하여설명한다.
발표구성.
1.
합성이필요한상황(스티커/오버레이, 배경교체, 모자이크/익명화, 데이터생성등)을예시로제시
하고설명한다.
2.
Poisson Blending을간단하게설명하고Normal_Clone과MIXED_CLONE 두가지를시각화하여비
교한다.
3.
추가적인합성방법을한가지더제시하여설명한후시각화한다.
Guide Code : Bitwise 연산을통한합성
